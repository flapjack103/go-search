## Running Go-Search

This assumes you have Go [installed](https://golang.org/dl/)

```
$ git clone https://github.com/flapjack103/go-search.git
$ cd go-search/
$ go build
$ ./go-search <your_go_project_path>
```

Open http://localhost:8080/main.html
